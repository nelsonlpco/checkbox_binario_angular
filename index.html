Checkbox binario angular

diretiva angular para criar entradas dinâmicas do tipo checkbox, com comportatamento de somatório binário.

View the Project on GitHub nelsonlpco/checkbox_binario_angular

Forma de uso

Importar o script 
 <script src="javascripts/CheckboxBinario.min.js"></script>
Injetar a diretiva no modulo 
<script type="text/javascript">
        angular.module("app",["checkboxBinario"])
          .controller('ctrl',['$scope',function($scope){
              $scope.valor = 9;   
          }]);
<script>
Aplicar a diretiva em um campo de entrada de texto 
  <checkbox-binario identificador="animais" modelo="valor" opcoes='C;C++;C#;Java' />

Parametros:
  • identidicador: Identificador do elemento.
  • modelo: variavel que receberá o valor da soma dos campos selecionado.
  • opcoes: Lista de opções que serão apresentadas ao usuário, as mesmas devem ser separadas pelo caractere ';'.
    Exemplo: "C;C++;C#;Java"

Desmontração

Marque as linguagens de programção que conhece.

Valor da somatória dos campos: {{valor}}.
